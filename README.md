# enhance

Plugin structure and tool use analyzer - validates plugin.json, MCP tools, and security patterns

## Installation

```bash
# Claude Code
claude mcp add-plugin agent-sh/enhance

# Or install from marketplace
agentsys install enhance
```

## Usage

```
/enhance
```
